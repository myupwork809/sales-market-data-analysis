# Sales & Market Research Data Analysis

This repository showcases sample projects demonstrating my skills in sales analysis, predictive modeling, and market research using Google Sheets.

## Projects Included

1. **Sales Trend Analysis**
   - Cleaned sales dataset, calculated trendlines and R², forecasted revenue.
   - Deliverables: Trendline chart, PDF report with insights.

2. **Customer Segmentation Analysis**
   - Segmented customers based on purchase behavior and demographics.
   - Deliverables: Segmentation charts, summary report with recommendations.

3. **Survey Data Analysis**
   - Analyzed customer survey responses to identify satisfaction drivers.
   - Deliverables: Response distribution charts, actionable insight report.

## Tools Used
- Google Sheets (Data Cleaning, Trendline & Pivot Tables, Charts)
- PDF Exports for reports

## How to Use
- Explore `datasets/` for sample data.
- View `charts/` for visual trends.
- Read `analysis_reports/` for detailed insights.
